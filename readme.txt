WA Chat WP Plugin adalah plugin WordPress untuk menambahkan tombol live chat WhatsApp pada situs Anda, memudahkan pengunjung untuk menghubungi Anda langsung melalui WhatsApp.

== Description ==

WA Chat WP Plugin memungkinkan Anda menampilkan tombol WhatsApp di situs web WordPress Anda. Dengan plugin ini, pengunjung situs dapat dengan mudah menghubungi Anda melalui WhatsApp hanya dengan satu klik.

Fitur utama:

* Menampilkan tombol WhatsApp di halaman situs web Anda.
* Mengatur nomor WhatsApp yang dituju.
* Mengubah teks tombol WhatsApp.
* Memilih posisi tombol (kanan bawah atau kiri bawah).
* Menyesuaikan warna tombol dan teks sesuai preferensi.

== Installation ==

1. Unggah folder plugin `wa-chat-wp-plugin` ke direktori `/wp-content/plugins/`.
2. Aktifkan plugin melalui menu 'Plugins' di WordPress.
3. Pergi ke 'Settings' -> 'WA Chat' untuk menyesuaikan pengaturan.

== Frequently Asked Questions ==

= Bagaimana cara mengubah nomor WhatsApp yang dituju? =
Pergi ke halaman pengaturan plugin di 'Settings' -> 'WA Chat', kemudian masukkan nomor WhatsApp yang ingin digunakan.

= Apakah saya dapat mengubah posisi tombol WhatsApp? =
Ya, Anda dapat memilih posisi tombol antara kiri bawah atau kanan bawah dari halaman pengaturan plugin.

= Bagaimana cara mengubah warna tombol? =
Anda dapat mengubah warna tombol dan teks dari halaman pengaturan plugin menggunakan pemilih warna yang disediakan.
